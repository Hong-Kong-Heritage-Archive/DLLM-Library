# 無大台香港典藏館 DLLM-Library

「無大台香港典藏館」以開源方式賦能海外香港社群，讓每個地區都能獨立運行屬於自己的文化保存平台。通過公開透明的代碼與去中心化的架構，我們希望每一本書、每件藏品都能找到它的歸宿，共同守護香港的文化記憶。

## 項目背景及目的

「無大台香港典藏館」是一個開源的社群平台，專為海外香港人設計，旨在促進珍貴圖書及文化藏品的保存與傳承。鑑於香港人移居海外後，個人藏品可能因空間限制或下一代興趣轉移而面臨丟棄風險，本項目希望通過去中心化的方式，讓全球香港社群：

- **List and Share** 個人珍藏的圖書或文化物品，與同好交流；
- **Borrow or Transfer** 藏品給真正珍惜的人，避免文化遺產流失；
- **Enable Communities** 不同地區的香港社群可自行搭建伺服器，靈活管理本地化平台。

作為開源項目，源代碼將公開於 GitHub，允許任何海外香港社群自由下載、修改及部署，確保「無大台」的精神貫徹，無需依賴單一中央伺服器。

## Phase 1 核心功能

Phase 1 專注於提供簡單穩定的功能，支援藏品數據上載、瀏覽及虛擬所有權轉移，並確保開源架構易於部署。核心功能如下：

**2.1 用戶個人檔案 (User Profile)**

- 用戶註冊並建立個人檔案，包含：
  - **Email Address**（唯一必填聯繫方式，保障私隱）
  - **Postcode**（僅顯示大致地區，如「Sydney, Australia」，精確地址由用戶自行決定是否分享）
  - 可選填：其他聯繫方式（如 Telegram、WhatsApp，公開程度由用戶控制）
- 用戶可選擇匿名或暱稱，保護個人資料。

**2.2 藏品上載及瀏覽 (Item Listing & Browsing)**

- 用戶可上載圖書或其他文化藏品的資料，包括：
  - **Item Name**（如《香港街頭故事》）
  - **Condition**（如「近全新」、「有輕微折痕」）
  - **Category**（如文學、歷史、攝影集，支援多語言標籤）
  - **Optional Details**（如出版年份、照片、藏品背後的故事）
- 平台提供基本的**Search & Filter**功能，按地區、類別或關鍵字查找藏品。
- 藏品標記為「可借閱」、「可交換」或「可贈送」，由上載者決定。

**2.3 虛擬所有權轉移 (Virtual Ownership Transfer)**

- 用戶可通過 App 標記藏品的「借閱」或「轉讓」狀態，紀錄交易意向。
- 平台僅提供**Virtual Confirmation**，實際聯繫及物流由雙方自行安排。
- 例如：用戶 A 標記「借出《九龍城寨》」，用戶 B 申請借閱，系統更新狀態並通知雙方，後續交收私下協調。

**2.4 語言及介面**

- App 支援**繁體中文**及**英文**雙語介面，適應海外用戶需求。
- 用戶可自由選擇上載資料的語言，系統自動生成中英雙語標籤（如「香港漫畫 / Hong Kong Comics」）。

## 開源運作模式

- **去中心化原則**：本項目不依賴單一伺服器，鼓勵各地區香港社群自行搭建獨立實例（Instance）。例如，倫敦社群可運行自己的伺服器，獨立管理本地用戶及藏品數據。
- **源代碼公開**：所有代碼（前端、後端及數據庫結構）將上載至 GitHub，採用 MIT License，允許自由使用、修改及分發。
- **部署靈活性**：提供詳細的**Deployment Guide**，包括 Docker 容器化設置，方便非技術用戶在雲端（如 AWS、Heroku）或本地伺服器運行。
- **私隱保護**：用戶資料（特別是地址）不公開，伺服器管理員需遵守基本數據保護指引（參考 GDPR）。
- **免費使用**：開源模式確保無需付費，社群可自行承擔伺服器運營成本，或通過捐款維持。

## 技術概要

- **Open Source Stack**：
  - **前端**：未定（簡潔雙語 UI，支援手機及桌面）
  - **後端**：未定，expose graphQL（輕量 API，易於擴展）
  - **數據庫**：PostgreSQL（開源且高效，支援多地區部署）
  - **容器化**：Docker（簡化伺服器設置，確保一致性）
- **GitHub Repository**：包含完整源代碼、安裝指引及 API 文檔，方便開發者貢獻。
- **Security**：採用 HTTPS 加密，確保用戶數據傳輸安全；伺服器管理員可自行配置額外安全措施。
- **UI/UX**：簡約設計，包含「上載藏品」、「瀏覽」及「我的檔案」三大核心分頁，易於本地化修改。

## 社群參與

- **貢獻方式**：歡迎全球開發者及香港社群參與，提交 Pull Requests 改進功能、翻譯介面或修復 Bug。
- **本地化支持**：提供多語言模板，方便社群將 App 翻譯成其他語言（如日語、德語）以融入當地香港人網絡。
- **討論渠道**：設立 Discord 或 Matrix 群組，供開發者及用戶交流技術問題及建議。

## 未來展望

Phase 1 將驗證開源模式的可行性，並收集社群反饋。未來可能的方向包括：

- 跨伺服器數據共享協議，實現全球藏品搜索；
- 整合文化故事功能，記錄每件藏品的獨特背景；
- 提供插件系統，讓社群自定義功能（如活動通知、藏品展覽）。
